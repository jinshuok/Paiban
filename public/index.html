开发文档
组织 ID（X-Tenant-Id）
default
所有接口调用时必须附带此请求头
API 密钥（X-Api-Key）
—
用于外部系统无登录态调用，仅限只读接口

GET 获取配置（成员、状态、分组）

GET /api/config X-Tenant-Id: default X-Api-Key: your-api-key
响应示例
{
  "groups": [{ "id": "g1", "name": "销售部" }],
  "members": [{ "id": "m1", "name": "张三", "uid": "zhangsan", "groupId": "g1" }],
  "statuses": [{ "id": "work", "label": "正常班", "short": "班", "color": "#2563eb" }],
  "clickCycle": ["work", "duty", "rest", null],
  "stats": [{ "countAs": "work", "label": "班", "color": "#2563eb" }]
}

GET 获取某月排班数据

GET /api/schedule/2026/4 X-Tenant-Id: default X-Api-Key: your-api-key
响应示例
{
  "m1-2026-4-1": "work",
  "m1-2026-4-2": "rest",
  "m2-2026-4-1": "duty"
}
键名格式：memberId-year-month-day，值为 statusId

POST 保存排班数据

POST /api/schedule/2026/4 Content-Type: application/json X-Tenant-Id: default X-Api-Key: your-api-key { "m1-2026-4-16": "duty", "m2-2026-4-16": "rest", "m1-2026-4-17": null }
传 null 可清除对应日期的排班；此接口需要密钥对应的租户具备写权限校验（登录态下可直接调用）
